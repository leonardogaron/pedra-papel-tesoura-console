# Pedra, Papel e Tesoura (Console - C#)

Projeto simples desenvolvido em C# com o objetivo de praticar lógica de programação, estruturas condicionais e interação com o usuário via console.

## Descrição

Este projeto simula o jogo clássico Pedra, Papel e Tesoura, onde o jogador escolhe uma opção e o computador gera uma jogada aleatória. O sistema compara as escolhas e exibe o resultado da partida.

## Funcionalidades

* Entrada de dados pelo usuário
* Validação de jogada inválida
* Geração de jogada aleatória para o computador
* Sistema de empate com repetição automática
* Exibição do resultado (vitória, derrota ou empate)
* Uso de switch expressions para identificar jogadas

## Tecnologias utilizadas

* C#
* .NET
* Aplicação Console

## Como executar

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/pedra-papel-tesoura-console.git
```

2. Acesse a pasta do projeto:

```
cd pedra-papel-tesoura-console
```

3. Execute o projeto:

```
dotnet run
```
